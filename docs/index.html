📦 របៀបដំឡើង (Installation)

Windows

Option 1: Batch File (ងាយស្រួល)

  1. ទាញយក install-windows.bat
  2. Right-click → Run as administrator
  3. រង់ចាំដំណើរការបញ្ចប់
  4. បិតនិងបើក Office ឡើងវិញ

Option 2: PowerShell

  1. ទាញយក install-direct-registry.ps1
  2. Right-click → Run with PowerShell
  3. បិតនិងបើក Office ឡើងវិញ

Mac

  1. ទាញយក install-mac.sh
  2. បើក Terminal
  3. ចូលទៅ Downloads folder: cd ~/Downloads
  4. ដំណើរការ: bash install-mac.sh
  5. បិតនិងបើក Office ឡើងវិញ
បញ្ជាក់: Add-in នេះដំណើរការទាំង Windows និង Mac។ សម្រាប់ Mac ប្រសិនបើ installer មិនដំណើរការ អ្នកអាចប្រើ web demo ដោយផ្ទាល់។

✨ មុខងារ (Features)

  • ប្រតិទិនតាមខែ ជាមួយថ្ងៃចន្ទគតិនៅក្នុងគ្រាប់កោសិកានីមួយៗ
  • បំលែងស្វ័យប្រវត្តិពីសុរិយគតិទៅចន្ទគតិ
  • រូបរាងជាច្រើន: ចន្ទគតិ, សុរិយគតិ, ទាំងពីរ, ពាក្យដើម
  • អានថ្ងៃពី Excel cell ដោយផ្ទាល់
  • បញ្ចូលចូល document ឬ copy clipboard

🛠️ តម្រូវការប្រព័ន្ធ (Requirements)

  • Windows 10/11
  • PowerShell execution
